# 自定义策略指南

内置的 `quick` 和 `deep` 策略不满足你的需求？你可以创建自己的总结策略。

## 策略的本质

一个"策略"就是一个 **System Prompt + 执行流程**，告诉 AI Agent 如何把字幕转成笔记。

它可以简单到一句话：

```
把下面的视频字幕总结成一份教程，面向初学者。
```

也可以复杂到多个步骤、多次 LLM 调用。

## 创建自定义策略

### 方式一：修改已有策略

1. 复制 `strategies/quick.md` 或 `strategies/deep.md`
2. 修改里面的 System Prompt 和 User Prompt
3. 使用时告诉 AI Agent："用我的策略文件 `my-strategy.md` 总结这个视频"

### 方式二：从头写

创建一个 Markdown 文件，包含你要 AI Agent 遵循的指令。至少需要：

```markdown
# 我的自定义策略

## 你的角色
你是一个 XXX。你的任务是...

## 输入
你会收到以下信息：视频标题、作者、字幕全文

## 输出格式
你的输出必须包含：
1. ...
2. ...
3. ...

## 规则
- 规则 1
- 规则 2
```

## 示例：英文字幕 → 中英文双语笔记

```markdown
# 双语笔记策略

## System Prompt

你是一个中英双语技术翻译和笔记整理助手。
你的任务是将英文视频字幕整理成中英对照的学习笔记。

## 输出格式

1. **# 标题** — 英文原标题 + 中文翻译
2. **## 概述** — 中英双语，各 2-3 句
3. **## 核心概念** — 表格形式：英文术语 | 中文翻译 | 解释
4. **## 关键段落** — 原文 quote + 中文翻译，逐段对照
5. **## 词汇表** — 视频中出现的生词/术语

## 规则
- 保留英文原文的精确表达
- 中文翻译力求准确，技术术语采用业界通用译法
- 不确定的翻译标注"(?)"
```

## 示例：视频 + 外部搜索交叉验证

```markdown
# 交叉验证策略

## 工作流程

1. 先通读字幕，列出所有关键概念和术语
2. 对每个关键概念，搜索最新的官方文档和权威资料
3. 对比视频内容和外部资料，标注：
   - ✅ 一致的部分
   - ⚠️ 有偏差的部分（标注偏差内容）
   - 🆕 外部资料有但视频没提到的补充信息
4. 生成"交叉验证笔记"，包含视频内容和外部验证结果
```

## 提示词设计技巧

1. **明确角色** — "你是一个 XXX" 能显著提高输出质量
2. **结构化输出** — 给出明确的标题层级和内容要求
3. **约束规则** — 列出 DON'T（不要编造、不要省略等）
4. **处理边界情况** — 告诉 AI 遇到无字幕/AI字幕/非技术内容时怎么做
5. **输出格式** — 指定 Markdown、JSON 还是其他格式
